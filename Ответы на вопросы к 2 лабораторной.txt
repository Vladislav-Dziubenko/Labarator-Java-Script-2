1.Какие методы массивов можно использовать для обработки объектов в JavaScript?
- Основные: map(), filter(), reduce(), forEach(), find(), some(), every(), sort().
Они позволяют трансформировать, фильтровать, агрегировать и искать данные в массиве объектов.
2. Как сравнивать даты в строковом формате в JavaScript?
- Нужно преобразовать строки в объекты Date и сравнивать их с помощью операторов <, >, ===.
Например:
new Date("2025-01-15") < new Date("2025-02-01") // true
3. В чём разница между map(), filter() и reduce() при работе с массивами объектов?
- map() создаёт новый массив, преобразуя каждый элемент (например, взять только описания).
- filter() возвращает массив элементов, которые удовлетворяют условию (например, только дебетовые транзакции).
- reduce() сворачивает массив в одно значение (например, сумма всех транзакций).
